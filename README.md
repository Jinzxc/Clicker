# Prototype PressClicker

A prototype of automation code built on Python. <br /><br />

| Version | Changes                         |
| ------- | -----------------------------   |
|   1.0   | Autoclicker                     |
|   1.1   | Clicker reader                  |
|   1.2   | File control with reader        |
|   1.3   | Presser and associated reader   |
|   1.4   | Clicker integration to presser  |

## Requirements
- Python
- pyautogui
- pynput
- folders named actions & kactions
